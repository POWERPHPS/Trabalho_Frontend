# Trabalho_Frontend
Trabalho acadêmico para o desenvolvimento Frontend versionado

Repositório criado para o desenvolvimento de um trabalho acadêmico, onde
o seu objetivo é a criação do frontend de um site chamado Neuro Abraço,
esse site tem como objetivo ajudar mães de crianças neurodivergentes
com dicas, opções e comunicação com outras mães, assim ajudando a
espalhar o conhecimento de cuidados com crianças com TEA.
